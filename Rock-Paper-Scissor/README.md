# Rock Paper Scissor

This is a simple command line game in python. The game works as follows:

1. The user is asked to enter their choice of rock, paper or scissor.
2. The computer randomly chooses one of the three options.
3. The result is determined by the following rules:
    - If the player chooses rock and the computer chooses scissor, the player wins.
    - If the player chooses scissor and the computer chooses paper, the player wins.
    - If the player chooses paper and the computer chooses rock, the player wins.
    - If the player and the computer choose the same option, it is a tie.
